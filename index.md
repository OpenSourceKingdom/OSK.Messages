# OSK.Messages
This library is an implementation of a general management system of the [pub-sub pattern](https://learn.microsoft.com/en-us/azure/architecture/patterns/publisher-subscriber). It provides the ability to send messages and integrate with a variety of message delivery systems like RabbitMQ as well as a delivery pipeline once messages are received and consumed within an application. 

## 📖 Documentation
For further information about the library, usage, etc., please visit the **[Docs Page](https://opensourcekingdom.github.io/OSK.Messaging/)** (or check the `/docs` folder).
